# Campus Shenanigans — Project Website

**Group 6 · COMP 3350 · Winter 2026**

## How to Run Locally

This site is plain HTML/CSS/JavaScript No build tools, no dependencies, no installation required.

**Step 1.** Clone or download the repository.

**Step 2.** Navigate into the `website/` folder.

**Step 3.** Open `index.html` in your browser.

- **macOS:** `open index.html`
- **Windows:** double-click `index.html` in File Explorer
